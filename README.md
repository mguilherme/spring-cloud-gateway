# Spring Cloud Gateway test
[Spring Cloud Gateway](https://cloud.spring.io/spring-cloud-gateway/) example.

### Configuration
Add `127.0.0.1  example.com` to your `/etc/hosts`

### Usage
* Eureka: `http://localhost:8761/`
* Service-1: `http://localhost:8080/service1`
* Service-2: `http://localhost:8080/service2`
* WS Application: `http://example.com:8080/`
